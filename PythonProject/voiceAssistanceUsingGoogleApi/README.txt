We need to install two things.
1. Speech Recognition.
2. pyAudio

How to install it?
1. Connect to Internet.
2. Open command prompt in windows  
3. Type "pip install SpeechRecognition" without quotes and press enter key.
4. It will install SpeechRecognition software.
5. Type "pip install pyAudio" in command prompt without quotes and press enter key.
6. Find the appropriate .whl file from "https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio"
	For example mine is 64 bit amd processor PyAudio-0.2.11-cp38-cp38-win_amd64.whl, and download it.
	For 32 bit use this file "PyAudio-0.2.11-cp38-cp38-win32.whl".
7. Go to the folder where it is download.
	For example C:\Users\vikram\Downloads\
8. Install the .whl file with pip.
	For example in my case: C:\Users\bvikram2\Downloads>pip install PyAudio-0.2.11-cp38-cp38-win_amd64.whl
9. It will install pyAudio software.
10. Thanks.
